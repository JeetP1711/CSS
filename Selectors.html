Selectors In CSS

A CSS selector selects the HTML element(s) you want to style.

Types Of Selectors:

  1. Simple selectors (select elements based on name, id, class)
  2. Combinator selectors (select elements based on a specific relationship between them)
  3. Pseudo-class selectors (select elements based on a certain state)
  4. Pseudo-elements selectors (select and style a part of an element)
  5. Attribute selectors (select elements based on an attribute or attribute value)

Right now we will only learn about basic simple selectors.

        Styling the elements in different ways and through different ways:
        1. Selecting by name:
        p{
            styles here.....
        }
        This p is a direct element from the html file.
        
        2. Selecting by id:
        #parol{
            styles here.....
        }
        This is the id selector from the file, which selects the element with id = "parol".
        
        
        3. Selecting by Class:
        .eco{
            styles here.....
        }
        This is the class selector from the file, which selects the element/s with class ="eco"
        
        
        4. Selecting specific class element:
        p.eco{
            styles here.....
        }
        This will just select the paras with class "eco".
        
        
        5. Universal Selector:
        *{
            styles here.....
        }
        This will select all the elements of the file.
        

        6. Grouping Selector:
        h1, p, pre{
            styles here.....
        }
        This will select the elements from h1, p and pre.
